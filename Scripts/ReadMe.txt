
"Scripts" is a collections of scripts in PHP and Python for the user as an example of different ways he/she can get the most of the Mercado Pago resources. Some of them are for: 
-Refunds.
-Get payments info
-Create payment buttons


In order to use most of the API calls, the user will need to access the credentials in: https://www.mercadopago.com/mla/account/credentials
